# LAMP Stack Installation

## Overview

Install Apache, MariaDB and PHP.

---

## Purpose

Provide hosting for wiki software.

---

## Requirements

- Debian
- sudo access

---

## Procedure

### Install Apache

```bash
sudo apt install apache2
```

### Install MariaDB

```bash
sudo apt install mariadb-server
```

### Install PHP

```bash
sudo apt install php
```

---

## Verification

Open browser.
